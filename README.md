###Building the Data Warehouses (Data engineering)

###Objectives 
Devolop a modern data warehouses using SQL Server to consolidate sales data, enalbing analytical reporting and informed decission-makimg 

###Specifications 
 - **Data Sources**: Import dat from two sources systems (ERP and CRM) provided as CSv files
 - **Data Quality**:Cleanse and resolve data quality issues prior to analyis .
 - **Intergration**:Combine both sources into a single, user friendly data model designed               for analytical queries.
 -**Scope**:Focus on the latest datsets only ; histroization of data is not required.
 -**Dpcumentation**:provide clear docuementation of the data model to support both business             stakeholders and analytics teams.
